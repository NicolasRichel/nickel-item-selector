# \<nickel-item-selector\>
(Built with Polymer 2.0)

An adaptable item selector designed to be simple and customizable.

```html
<nickel-item-selector items="[1,2,3]">
  <input slot="item-elem" type="checkbox"/>
</nickel-item-selector>
```
