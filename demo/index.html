nickel-item-selector demo

Below is a simple example of how to use the nickel-item-selector element.
Note that the input element is slotted as "item-elem", i.e. the element that materialize the items and is used to select them.

<nickel-item-selector items="[1,2,3]">
   <input slot="item-elem" type="checkbox"/>
</nickel-item-selector>
